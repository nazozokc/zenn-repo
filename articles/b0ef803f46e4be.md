## CLI
### editor
#### neovim
エディターはneovimを使い続けています。最近だと、ほかのエディター使ってみよって使ってみてもついカーソル移動のときに**hjkl**使ってしまったり、終了の時:wqaってやってしまったりして、もうneovimから抜け出せなくなってしまいました、でも楽しかったらＯＫです！

### terminal 
#### wezterm
ターミナルはweztermです。weztermの何がいいって設定をluaでかけることですね、neovimとごっちゃになりそうですが、AIでほとんど設定してるのでOKです。あと単純に設定しすぎてしまいこれもまた抜け出せなくなりました。

### git操作
#### git
これは言わずもがなほとんどの人が使ってるやつですね

#### ghq
githubのレポジトリはghqを使ってひとまとめで管理できるようにしています。

### package manager
#### Nix
パッケージマネージャーはnixです。nixの何がいいって、一コマンド実行するだけで前の環境や、作成者と同じ環境が手に入ることだと思っています。あと、パッケージもファイル管理なので、今何が入ってるか一目でわかりやすいから使ってますね。

### ディレクトリ移動
#### zoxide
ディレクトリ移動にはzoxideを使用しています、zoxideでよかったことはフルパス入力しなくても`z <ディレクトリ名>`で一瞬で移動できるので、重宝しています。フルパス打っていた時代が信じられない、、、、

### ファイル操作
#### yazi
ファイル操作はyaziを使用しています。vimライクな操作感で操作することができるのでたまに使用していますが、基本的にはneovimのoil.nvimっていうファイル操作のプラグインで完結することが多いです。
