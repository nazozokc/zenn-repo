## Repository Overview

This is a zenn.dev repository for the user `nazozokc`. Articles are stored in the `articles/` directory as markdown files with frontmatter.

## Article Structure

Articles use Zenn markdown format with YAML frontmatter:

```markdown
---
title: "Article Title"
emoji: "emoji"
type: "tech" # or "idea"
topics: ["tag1", "tag2"]
published: true
published_at: "2026-01-30 20:55" # optional
---
```

## Directory Structure

- `articles/` - Published and draft Zenn articles (`.md` files)
- `books/` - Unused (contains only `.keep` file)
